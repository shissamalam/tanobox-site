Architecture

Self-hosted. Your data stays on your machine.

No cloud sync. No data leaving your building. All computation runs locally — a genuine feature for firms handling sensitive client financials.

⚙
Custom Calculation Engine
Purpose-built TypeScript formula engine. DAG-based dependency resolution with iterative circular reference solving. Built with Claude Code. Zero third-party formula library dependencies — no GPL licensing exposure.
🗄
PostgreSQL + Next.js Stack
Full 3-statement modeling stack. NUMERIC(20,4) precision for all financial values — never floating-point. Drizzle ORM, tRPC, shadcn/ui, AG Grid Community (MIT). QuickBooks Online + Desktop integration.
🪟
Windows Native Delivery
Ships as a pre-configured Windows 11 mini PC or a single PowerShell installer. WSL2 + Ubuntu + PM2. Opens in a browser tab. No cloud account, no Docker knowledge, no IT department required.

Risk Register

Known risks. Documented responses.

Every risk in the full business plan has a concrete mitigation. The four highest-priority items:

Calculation engine produces incorrect financial outputs
Critical
50+ unit tests before any customer sees the product. Balance sheet must balance to zero. CFS must reconcile in all scenarios. Explicit EULA disclaimer. "Verify with your CPA" language in product UI.
Windows Update breaks WSL2 on customer machine
High
Repair script on desktop at install time. Weekly WSL2 auto-export to C:\TanoBox\backup.tar.gz. Group Policy defers feature updates 365 days. One-click recovery in 3 minutes.
QuickBooks changes API requirements mid-year
Medium
QB Desktop SOAP/qbXML integration as fallback — no OAuth required, no Intuit approval. CSV import as universal fallback. Developer blog monitored weekly. May 2026 CloudEvents migration already tracked.
Founder capacity collapse across build + sell + support
High
Hard cap at 5 customers until support model is proven. Do not scale sales beyond what support can absorb. Partners not activated until solo validation produces 5 paying customers with documented close rates.
